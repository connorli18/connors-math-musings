<div class='section'>Relations</div><div class='subsection'>Definitions</div><b>[1] Relation:</b> $R$ is a relation from set $A$ to set $B$ provided that $R \subseteq A \times B$ (cartesian product).<br><b>[2] Inverse Relation:</b> Inverse relation of $R$, defined from $B$ to $A$, is defined as following, $R^{-1} = \{(y.x)\;|\;(x,y) \in R\}$<div class='subsection'>Visual Representations</div><b>[1] Digraphs:</b> directed graphs that represent the relation with a set of nodes and edges<br><b>[2] Boolean Matrix:</b> rows are elements of $A$, columns $B$, $1$ if there is a relation between $(x,y)$, otherwise $0$<div class='subsection'>Properties of Relations</div><b>[1] Reflexivity:</b> $\forall x \in A\;\;x\;R\;x$ <br><b>[2] Irreflexive:</b> $\forall x \in A\;\;x\;\cancel{R}\;x$<br><b>[3] Symmetric:</b> $\forall x,y \in A\;\;(x\;R\;y \implies y\;R\;x)$<br><b>[4] Antisymmetric:</b> $\forall x,y \in A\;\;(x\;R\;y \implies x = y)$<br><b>[5] Transitive:</b> $\forall x,y,z \in A\;\;(x\;R\;y \wedge y\;R\;z \implies x\;R\;z)$<br><b>[6] Equivalence Relation:</b> $R$ is reflexive, symmetric, and transitive!<div class='subsection'>Equivalence Classes</div><b>Definition:</b> If $a \in A$, the equivalence class of $a$, $[a]$, is denoted by $[a] = \{x \in A\;|\;x\;R\;a\}$<br><b>Properties:</b> We know that $(1)\;a \in [a]$, and $(2)\;a \neq \varnothing$, and $(3)\;\bigcup_{a \in A} [a] = A$.<div class='subsection'>Congruence Modulo N</div><b>Definition:</b> $x,y \in \mathbb{Z}$ are congruence-modulo n $(x \equiv y(\text{mod}\;n))$ if $n|(x-y)$. For example, $3,13$ are CM $5$ but $3,16$ are not!<div class='section'>Functions</div><div class='subsection'>Definitions</div><b>Definition:</b> $f: A \rightarrow B \subset A \times B$, each element $a \in A$ appears exactly once as a first element. Or, $\forall a \in A\;\exists! b \in B\;\;b = f(a)$<br><b>Terminology:</b> $A$ is the domain of $f$, $B$ is the co-domain/image0, $b$ (image, could be a set!) and $a$ (pre-image)<br><b>Bipartite Graphs:</b> Vertices join elements of $A$ (domain) to elements of $B$ (co-domain).<div class='subsection'>Properties of Functions</div><b>[1] Onto/Surjective Functions:</b> $f:A \rightarrow B$ provided ever element of $B$ has a pre-image.$$\;\forall b \in B,\;\exists a \in A,\;f(a) = b$$<b>[2] 1-to-1/Injective Functions:</b> $f$ provided that no element of $B$ is the image of $2$ distinct elements of $A$ using $f$.$$\forall a_1,a_2 \in A\;\;f(a_1) = f(a_2) \implies a_1 = a_2$$<b>[3] Bijective Functions:</b> If function is both surjective AND injective.<div class='subsection'>Compositions of Functions</div><b>Definition:</b> Let $f: A \rightarrow B$ and $g: C \rightarrow D$, where $\text{im}(f) \subset C$. Then, $g \circ f: A \rightarrow D$ is defined as $g \circ f = g(f(x))$ <br><b>Properties:</b> (1) Compositioning $2$ onto/inj/bij functions$\implies$onto/inj/bij function<div class='subsection'>Inverse of Functions</div><b>Definition:</b> For function $f: A \rightarrow B$, if there exists some $g: B \rightarrow A$ such that $g \circ f = f \circ g = I$, then $g = f^{=1}$.<br><b>Theorem:</b> If $f^{-1}$ exists, then $f$ is a bijection and the inverse is unique (iff).<div class='section'>Pigeonhole Principle</div><div class='subsection'>Definitions</div><b>[1] PHP:</b> If we have $n$ pigeons living in m holes with $n > m$, then at least one of the holds contains two or more pigeons. Also, for any function $f: A \rightarrow B$, if $|A| > |B|$, then $f$ cannot be injective.<br><b>[2] Generalized PHP:</b> If we have $n$ pigeons living in $m$ holes with $n \geq m$, then there is at least one hole containing at least $\lceil\frac{n}{m} \rceil$ pigeons. The ceiling functions is always defined by $x \leq \lceil x \rceil < x+1$.<div class='subsection'>Examples</div><b>[1] Example:</b> $A = \{1,2,\dots,8\}$. If $5$ integers are selected from $A$, we must have at least one pair with sum $9$. There are four pairs of numbers that add to $9$, since we pick $5$ numbers, then our fifth number must complete a pair. Using the generalized PHP, we have $\lceil \frac{5}{4} \rceil = 2$ of the 5 numbers that sum up to $9$.<br><b>[2] Example:</b> In a group of $n$ people, at least $2$ people will have the same number of friends. Each person can have at most $n-1$ friends, which means that $\lceil \frac{n}{n-1} \rceil = 2$ people have the same number of friends.<div class='subsection'>PHP for Functions</div><b>Definition:</b> A set is <u>countable</u> iff there exists a bijection between the elements of the set and $\mathbb{N}$.<br><b>[1] Theorem:</b> If $f$ is one-to-one, then $|A| \leq |B|$. Or, if $|A| > |B|$, then $f$ is not one-to-one.<br><b>[2] Theorem:</b> If $f$ is onto, then $|A| \geq |B|$. Or, if $|A| < |B|$, then $f$ is not onto.<div class='section'>Advanced Proofs</div><b>[1] Weak Induction:</b> Base Case, Inductive Hypothesis, Inductive Step<br><b>[2] Smallest Counter-example:</b> Base Case, Smallest Counterexample, Contradiction<br><b>[3] Strong Induction:</b> Base Case, Inductive Hypothesis (all cases up to $n$). Since every $\mathbb{Z}$ is even or odd, $0\leq r < 2$. <br><b>[2] Theorem Modular:</b> We know that $a \equiv b\;\text{mod}\;n \iff a\;\text{mod}\;n = b \;\text{mod}\;n$.$$n|(a-b) \iff a \equiv b\;\text{mod}\;n \iff \exists k \in \mathbb{Z},\;a = b+kn$$<b>[3] Greatest Common Divisor:</b> GCD is unique, except where $a,b = 0$. We also know $\text{GCD}(a,b) = \text{GCD}(b,a)$.<br><b>[4] Prime Numbers:</b> The set of prime numbers if infinite. <br><b>[5] Fundamental Theorem of Arithmetic:</b> If $n \in \mathbb{Z}^+$, there exists a factorization of $n$ into a product of powers of primes.<div class='subsection'>Greatest Common Divisor</div><b>[1] Euclid's Observation:</b> We know that $\text{GCD}(a,b) = \text{GCD}(b, a\;\text{mod}\;b)$<br><b>[2] Modular Operators:</b> Considered over the set $\mathbb{Z}_n = \{0,1,\dots,n-1\}$. EX: $a \oplus b = a + b\;\text{mod}\;n$ or $a \otimes b = a \times b\;\text{mod}\;n$<br><b>[3] Modular Equivalence P2:</b> We know that following property holds:$$ab \equiv (a\;\text{mod}\;n)(b\;\text{mod}\;n)\text{mod}\;n\;\;\text{   AND   }\;\;a^m \equiv (a\;\text{mod}\;n)^m\;\text{mod}\;n$$<b>[4] Modular Reciprocal:</b> $a$ is the modular reciprocal of $b \in \mathbb{Z}_n$ if $a \otimes b = 1$. Also, $a$ is invertible iff $\text{GCD}(a,n) = 1$.<div class='subsection'>Extended Euclid Algo:</div> Find the smallest combination of 431a + 29b. First, find the GCD(431,29). Then, work backwards to plug in our equivalences.$$\begin{cases}431 = 29 \times 14 + 25 \\29 = 25 \times 1 + 4 \\25 = 4 \times 6 + 1 \\\end{cases} \implies\begin{cases}25 = 431 - 29 \times 14 \\4 = 29 - 25 \times 1 \\1 = 25 - 4 \times 6 \\\end{cases} \implies\begin{cases}4 = 29 - (431 - 29 \times 14) \times 1\\4 = 29 \times 15 - 431 \\1 = (431 - 29 \times 14) - (29 \times 15 - 431) \times 6 \\1 = 431 \times 7 - 29 \times 104\end{cases}$$If we wanted to find the inverse of $x = 29$ in the space $\mathbb{Z}_{431}$, we use the algorithm to find its corresponding coefficient (-104). Turning this into a positive integer, we have $(-104 + 431) = 327$, our inverse.<div class='section'>Counting</div><div class='subsection'>Definitions</div><b>[1] Permutations:</b> Order matters, $P(n,k) = \frac{n!}{(n-k)!}$<br><b>[2] Combinations:</b> Order doesn't matter, $C(n,k) = \frac{n!}{k!(n-k)!}$<br><b>[3] Binomial Theorem:</b>$$(x+y)^n = \sum_{k=0}^n {n \choose k} x^{n-k}y^k$$<b>[4] Pascal's Identity:</b>$${n \choose k} = {n-1 \choose k-1} + {n - 1 \choose k}$$<div class='subsection'>Inclusion-Exclusion Principle</div>Note that the coefficients for these terms is often denoted by the number of combinations ${n \choose k}$ where $n$ is total number of groups and $k$ is the total number of groups selected as part of that.$$\left|\bigcup_{i = 1}^{n} A_i\right| = \sum_{i=1}^n |A_i| - \sum_{1 \leq i < j \leq n} |A_i \cap A_j| + \sum_{1 \leq i < j < k \leq n} |A_i \cap A_j \cap A_k| - \dots + (-1)^{n-1} |A_1 \cap \dots \cap A_n|$$<div class='section'>Graphs</div><div class='subsection'>Definitions and Terminology</div><b>[1] Graph:</b> Graph $G = (V,E)$ consists of two sets, $V$, set of vertices, and $E$, a set of edges.<br><b>[2] Edge:</b> An edge joins two vertices called its endpoints, denoted $\{u,v\}$ or $uv$. <br><b>[3] Simple Graph:</b> A simple graph is an undirected graph with not loops and no parallel edges.<br><b>[4] Adjacency:</b> Two vertices are adjacent if there is an edge joining them, also to say they are neighbors.<br><b>[5] Adjacency matrix:</b> A matrix of the graph $G = (V,E)$ on $V \times V$ such that:$$A_{ij} = \begin{cases}1 & \text{if there is an edge connecting $v_i$ and $v_j$} \\2 & \text{otherwise, no edge}\end{cases}$$<b>[6] Neighborhood:</b> Set of all neighbors of $v$ such that $N(v) = \{u \in V | \{u,v\}\}$<br><b>[7] Degree:</b> Number of neighbors of a vertex is called the degree, $d(v) = |N(v)|$. The max degree of graph $G$ is denoted $\Delta(G)$ and minimum is denoted $\delta(G)$.<br><div class='subsection'>Handshaking Theorem</div><b>Definition:</b> The sum of the degrees of each vertex in a graph $G = (V,E)$ is equal to twice the number of edges. This also applies to non-simple graphs, except that loops would add degree two!$$\sum_{v \in V} d(v) = 2|E|$$<div class='subsection'>Types of Graphs</div><b>[1] Complete Graph:</b> A graph with every pair of vertices joined by an edge. $K_n$ denotes this on $n$ vertices.<br><b>[2] Path Graph:</b> A path graph has vertices $\{v_1,v_2,\dots,v_n\}$ and edges $\{e_1,e_2,\dots,e_{n-1}\}$ such that edge $e_k$ joins $\{v_k,v_{k+1}\}$<br><b>[3] Cycle Graph:</b> Cycle graph is like a path graph except $\{v_1,v_n\}$ is connected also by an $\{e_n\}$.<br><b>[4] Bipartite Graph:</b> A graph with a vertex set which can be partitioned into two disjoint sets. Every edge joins a vertex in in $A$ with a vertex in $B$.<br><b>[5] Regular Graph:</b> A graph is regular provided every vertex has the same degree.<div class='subsection'>Isomorphism</div><b>Definition:</b> Two graphs are isomorphic iff there exists a bijective fnction representing an isomorphism of $G$ to $H$.<br><b>Proving non-Iso:</b> (1) Number of vertices (2) number of edges (3) degrees (4) adjacencies (5) cycles (6) connected<div class='subsection'>Properties of Graphs</div><b>[1] Subgraph:</b> A subgraph $H$ is a graph contained in another graph $G$. <br><b>[2] Spanning Subgraph:</b> $H$ is a spanning subgraph of $G$ if $V(G) = V(H)$ and $E(H) \subseteq E(G)$.<br><b>[3] Clique:</b> Subset of vertices $S \subset V(G)$ is a clique if any two distinct vertices are adjacent. Cliques of size $0,1$ also exist.<br><b>[4] Independent:</b> Subset of vertices S is called independent provided no two vertices in S are adjacent. <br><b>[5] Complement:</b> The complement is defined as follows: $V(\overline{G}) = V(G)$, but $E(\overline{G})$ connects all the vertices not connected and disconnects all previously connected. $E(\overline{G}) = \{uv|u,v \in V(G), u \neq v, uv \notin E(G)\}$ <br><b>[6] Walk:</b> Sequence of vertices where each vertex is adjacent to the next vertex. <br><b>[7] Path:</b> Walk in which no vertex is repeated. If there is a walk, then there is a path from $x$ to $y$. <br><b>[8] Hamiltonian Path:</b> A path containing all the vertices of a graph.<br><b>[9] Connection:</b> We say that $u$ is connected to $v$ iff there is a $(u,v)$-path in $G$. <br><b>[10] Connected Graph:</b> A graph where each pair of vertices is connected by a path.
